Summary of Latitude vs X




The purpose of this project was to analyze how weather changes in relation to latitude. To accomplish this, we pulled data from the OpenWeatherMap API on +500 cities. After assembling the data, we used MatPlotLib to plot four different factors vs Latitude. The four factors were temperature, cloudiness, windiness, and humidity. This site provides the source data and visulaizations created as part of the analysis, as well as a key observation for each factor.
